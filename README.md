Rural Loan Dataset Analysis using Python

This project analyzes a rural loan dataset to identify key patterns in borrowing behavior and household characteristics. The dataset contains over 1,000 loan records, which were cleaned and preprocessed using Python’s Pandas library. Data cleaning involved handling missing values, detecting and removing outliers using the Interquartile Range (IQR) method, and refining the dataset through column pruning to improve analysis efficiency.

Exploratory Data Analysis (EDA) was conducted to understand relationships between variables, particularly focusing on how loan amounts vary with household size. GroupBy aggregations were used to derive meaningful insights and summarize trends across different categories.

To enhance interpretability, data visualizations were created using Matplotlib and Seaborn. Bar charts and pie charts were used to illustrate trends related to housing conditions and infrastructure availability in rural areas. These visual insights help in understanding socio-economic factors influencing loan distribution.

The findings from this analysis were compiled into a structured report, highlighting key observations and patterns in rural financing. This project demonstrates practical skills in data cleaning, transformation, analysis, and visualization, making it useful for understanding real-world financial datasets.

Tech Stack: Python, Pandas, Matplotlib, Seaborn, Data Cleaning, EDA, Data Visualization
